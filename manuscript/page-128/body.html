Fig. 96. Saw-Vise.

To file and set a saw, the saw is first fastened in \ , which is held in the right hand and its point in the thumb and fingers of the left. Pressure is applied only on the forward stroke, which should be long and even, the file being raised above the tooth on the return stroke. The file should cut in the direction of the set, that is, the teeth having the set away from the worker are filed first. Every alternate tooth, 1st. 3d, 5th, etc., is filed, and then the saw is reversed and the other set, the 2nd, 4th, 6th, etc., is filed.
