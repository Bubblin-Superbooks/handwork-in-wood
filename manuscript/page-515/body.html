There are two principal methods of gluing edge-to-edge joints, rubbing and squeezing. In a rubbed joint, the surfaces to be joined should be planed so as to meet thruout exactly. After properly planing one edge of each board, keep one board in the vise, jointed edge up, and place its to-be neighbor in position upon it. Then use these four tests for an exact fit. (1 Sight down the end to see that the faces lie in the same plane. (2 Examine the crack from both sides. Be sure that both ends touch. Test this by pulling down hard on one end of the upper board and noticing if the other end is still in contact. If the other end opens, swing the upper board horizontally on the lower board to see where the high place is and then correct it.
